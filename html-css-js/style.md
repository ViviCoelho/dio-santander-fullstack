<h1>CSS</h1>
Funciona como uma cascata
<h3>Seletores</h3>
seletores => declarações

ID X Classe
representam quaisquer tipos de elementos

id => # só pode ser usado uma vez

class => . pode ser usado várias vezes


<h3>BOX MODEL<h3>
- margin => espaçamento entre elementos
- border => circundam o padding e o conteudo
- padding => espaçamento entre a borda e conteúdo
- content => conteúdo

<h3>Estilizando elementos</h3>
margin e padding:
exemplo: post{
    padding: 10px 15px 12px 0; (superior / direita / inferior / esquerda)
}

background ou background-color (mudar a cor de fundo);

border 
- largura
- cor
- estilo
border-radius: arredonda as bordas


<h3>Estilizando texto</h3>
font-style => normal, italico.
font-weigth => peso do textp: normal, bold.

text-transform: uppercase (todas maiusculas)/ lowercase (todas minusculas) / capitalize(coloca todas as primeiras letras em maiusculo)

text-decoration: underline(linha embaixo) / overline(linha em cima) / line-through (traçado) / none(nenhum);

<h3>Estilizando lista</h3>
list-style-type => square / upper-roman / / \"1F44D"

alterar imagem da ul list-style-img: url("rocket.png")

<h3>Propriedade e dimensões</h3>
width => largura
height => altura

max-width => largura maxima que os elementos podem ter
max-height => altura maxima que os elementos podem ter

text-aling => alinhar texto
