<h1>Elemento HTML</h1>
h1 class="titulo">Titulo/h1

onde: 
h1 => abre a tag
class="titulo" => é um atributo
Tìtulo => é o conteúdo
/h1> => fecha a tag

<h3>Estrutura básica</h3>
!DOCTYPE html> => apenas diz ao navegador o que está escrevendo <br>
html> => tudo que diz respeito ao documento da página está neste elemento<br>
head> => algumas metainformações<br>
    meta> => usada para várias coisas<br>
    title></title> => título na aba do navegador
/head><br>
 body> => onde está o conteúdo da página
  </body>
</html>

<h3>Semântica</h3>
versão 5 <br>
section> => representa uma seção genérica de conteúdo<br>
header> => cabeçalho<br>
article> => representa um conteúdo relevante dentro da página como um artigo de um blog<br>
aside> => conteúdo relacionado a página principal<br>
footer> => rodapé<br>
h1-h6> => não foram criados na versão 5. Só pode haver um H1 por página.<br>

<h3>Tags para texto</h3>
h1>=> titulo da página</h1> <br>
h2>=> titulo da seção</h2> <br>
h3>=> titulo do artigo</h3> <br>
p> => conteudo do artigo (pode ser adicionado, fotos videos, texto entre outros.</p>
